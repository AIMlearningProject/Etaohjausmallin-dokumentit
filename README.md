# Etäohjausmallin-dokumentit

Tämä repository pitää sisällää AIMlearning-hankkeessa kehitetyt dokumentti- ja raporttipohjat, jotka ovat kaikkien vapaassa käytössä. Raportteja ja dokumentteja voidaan päivittää hankkeen aikana uudempiin versioihin. Alapuolelta löydät myös latauslinkit hankkeessa kehitettyihin palautelomakkeisiin (suomeksi ja englanniksi). Raportti- ja dokumenttipohjat ovat osa hankkeessa kehitettyä etäohjauksen opasta, joka julkaistaan loppuvuodesta 2025. Hanke on Euroopan unionin osarahoittama ja toteutetaan ajalla 01.02.2024-30.04.2027. Lisätietoa AIMlearningistä löytyy [https://aimlearning.fi/](https://aimlearning.fi/)

This repository contains document and report templates developed in the AIMlearning project, which are freely available to everyone. Reports and documents may be updated to newer versions during the project. The reports are part of a remote guidance guide developed in the project, which will be published at the end of 2025. The project is co-funded by the European Union and will be running from 1.2.2024 to 30.4.2027. For more information about AIMlearning visit [https://aimlearning.fi/en/](https://aimlearning.fi/en/)
